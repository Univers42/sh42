title: Getting started
---

1. Install dependencies:

    `npm install`

2. Start the local dev server:

   - npm run start

3. Edit docs:

   - Add or edit files under `docs/`. The sidebar is autogenerated.
   - Blog content is under `blog/`.

4. Build the site for production:
   - npm run build
   - Preview the build:
     - npm run serve
